# Adatvédelmi szabályzat

A Trace kabala **fel van szerelve egy AirTagg**, amely nyomon követi a helyzetét. Ha van egy iPhone-ja, akkor kaphat egy üzenetet, amely arról tájékoztatja, hogy egy nem önhöz tartozó AirTag van a közelében.

A Trace viselőinek **névtelenségének és magánéletének** tiszteletben tartása érdekében a weboldalon megjelenített pozíciót **véletlenszerűen** generáljuk a kabalafigura pozíciójának 2 km-es körzetében\*\*.

Az európai RGPD-rendeletekkel összhangban Ön kérheti a helymeghatározási adatok törlését. Küldjön egy e-mailt a<a href="mailto:traceacrosstheworld@gmail.com"></a> címre, megjelölve azt az időszakot, amikor a Trace kabalaállat az Ön birtokában volt.
