# Privatleben

Das Maskottchen Trace ist **mit einem AirTag** ausgestattet, um seine Position zu verfolgen. Wenn Sie ein Iphone besitzen, kann es sein, dass Sie eine Nachricht erhalten, in der erklärt wird, dass sich ein AirTag, der Ihnen nicht gehört, in Ihrer Nähe befindet.

Um **die Anonymität und Privatsphäre** der Trace-Träger zu wahren, wird die auf der Website angezeigte Position **zufällig in einem Umkreis von 2 km** um die Position des Maskottchens generiert.

Im Rahmen der EU-Verordnung DSGVO können Sie die Löschung von Standortdaten verlangen. Senden Sie eine E-Mail an die<a href="mailto:traceacrosstheworld@gmail.com">Kontaktadresse</a> und geben Sie den Zeitraum an, in dem Sie das Trace-Maskottchen besessen haben.
