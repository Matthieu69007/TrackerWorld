# Präsentation

Willkommen beim Projekt "Trace across the world".

### Ziel

Das Maskottchen Trace ist mit **einem Ortungssystem** ausgestattet, mit dem seine Spur rund um den Globus verfolgt werden kann. Im Oktober 2023 startet **Trace in Lyon, Frankreich, und macht eine Weltreise**, indem es **von Hand zu Hand zwischen den Trägern** weitergereicht wird, die es auf die Reise schicken.

### Wie funktioniert das?

Die Route von Trace ist festgelegt und **führt durch symbolische Etappenstädte** rund um die Welt.

Achtung, Trace hat beschlossen, diese lange Reise **ohne Flug zu machen.** Sie müssen **mit dem Schiff** die Ozeane überqueren!
