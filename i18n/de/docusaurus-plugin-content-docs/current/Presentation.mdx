# Präsentation

Willkommen beim Projekt "Trace across the world".

### Ziel

Das Maskottchen Trace ist mit einem Ortungssystem ausgestattet, mit dem man seine Spur rund um den Globus verfolgen kann. Trace ist am XX/XX/2023 in Lyon in Frankreich gestartet und macht eine Weltreise, indem sie Träger von Hand zu Hand weitergibt, die sie auf Reisen schicken.

### Wie funktioniert das Ganze?

Die Route von Trace ist festgelegt und führt durch symbolische Etappenstädte rund um die Welt.

Achtung: Trace hat sich entschieden, diese lange Reise ohne Flugzeug anzutreten. Du musst ein Schiff nehmen, um die Ozeane zu überqueren!
