# 隐私政策

Trace 吉祥物**配备了一个 AirTag**，用于跟踪其位置。 如果你拥有 iPhone，你可能会收到一条信息，说明你附近有一个不属于你的 AirTag。

为了**尊重 "吉祥物追踪 "佩戴者的匿名性和隐私**，网站上显示的位置是在吉祥物位置半径 2 公里**范围内**随机生成的。

根据欧洲 RGPD 法规，您可以要求删除位置数据。 发送电子邮件至<a href="mailto:traceacrosstheworld@gmail.com"></a> ，说明您拥有 Trace 吉祥物的时间段。
